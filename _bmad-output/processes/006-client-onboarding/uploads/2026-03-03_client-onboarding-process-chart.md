# Client Onboarding Process Chart (Simulated Upload)

## Process Flow

1. Client Inquiry Received (Relationship Manager) → CRM
2. Collect Client Documentation (Relationship Manager) → Document Portal
3. Verify Identity Documents (Ops Analyst) → Document Portal, ID Verification System
4. KYC/AML Screening (Compliance Officer) → KYC Platform
   - Decision: Pass/Fail
   - If Fail → Enhanced Due Diligence or Reject
5. Risk Assessment & Categorization (Compliance Officer) → KYC Platform
6. Account Setup & Configuration (Ops Analyst) → Core Banking
7. Welcome Pack & Access Provisioning (IT Support, Relationship Manager) → Core Banking, Email
8. Account Activation & Go-Live (Relationship Manager) → CRM, Core Banking

## Roles
- Relationship Manager
- Operations Analyst
- Compliance Officer
- IT Support

## Systems
- CRM (Salesforce)
- Document Portal (internal)
- ID Verification System (third-party)
- KYC Platform (internal)
- Core Banking System
